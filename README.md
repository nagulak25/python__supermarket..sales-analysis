# python__supermarket..sales-analysis
This project analyzes supermarket sales data using Python, Pandas, Matplotlib, and Seaborn. The goal is to explore sales trends, customer behavior, and product performance through data analysis and visualization.

The dataset includes information such as product lines, quantity, total sales, customer gender, branch location, payment methods, and ratings.

The project performs:

Data loading and exploration

Data cleaning and feature extraction

Sales statistics calculation

Product and branch performance analysis

Customer behavior analysis

Data visualization using charts and heatmaps

The analysis helps identify top-selling products, peak sales times, payment preferences, and branch performance, providing useful insights for business decision-making.

Tools Used: Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook.
